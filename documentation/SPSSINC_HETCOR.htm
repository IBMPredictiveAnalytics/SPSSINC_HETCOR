Categorical Correlations

This procedure calculates correlations for nominal, ordinal, and scale variables, accounting for the measurement levels of the variables. The procedure uses the hetcor function from the R polycor package.

To run this procedure:

► Select the variables to be correlated. Note that if a pair of variables is too far from being approximated by a bivariate normal distribution, the calculations may fail to converge.

Optionally, you can:

• Specify the handling of missing values. By default, each correlation is computed from all available data. Select Delete missing values listwise to specify that cases with missing values are omitted from the analysis.

• Specify the content of the output. By default, the output includes a table of correlations. You can specify whether the sample size and the correlation standard error are included in this table. Include table of correlation types produces a separate table indicating the type of correlation calculated for each pair of variables ("Pearson", "Polyserial", or "Polychoric").

• Specify the estimation method. Twostep can be considerably faster and is the default. Maximum likelihood appears more likely to suffer convergence failure.

Save Subdialog

Save R Implementation Code. The R implementation code used to perform the analysis is written to the specified file, and may be useful as a starting point for extending the analysis.

Calculate correlations. When saving the R implementation code, you may want to uncheck this box. This results in saving the implementation code without running the analysis.

Additional Features

This dialog generates syntax for the SPSSINC HETCOR extension command. To display help for this command, run the following syntax:

SPSSINC HETCOR /HELP.

Requirements

This dialog requires the Integration Plug-In for Python, the Integration Plug-In for R and the R polycor package. For more information, see How to Get Integration Plug-Ins, available under Core System > Frequently Asked Questions in the IBM® SPSS® Statistics Help system.
